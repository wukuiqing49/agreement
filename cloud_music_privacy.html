Cloud Music 隐私政策

生效日期：2025-12-15

感谢您使用 Cloud Music。本隐私政策说明我们如何收集、使用和保护您的信息。

1. 信息收集

  • 本地媒体访问：应用需要访问您设备上的音频文件（本地存储或云盘），用于显示列表和播放音乐。我们仅读取用户选择播放的音频文件，不会上传或分享这些文件。
  • 广告信息：应用包含第三方广告 SDK（如 AdMob 或其他广告平台）。广告 SDK 可能收集匿名信息（如设备类型、广告点击情况）用于展示广告。我们不收集个人身份信息用于广告。
  • 性能信息：我们可能收集应用崩溃日志或错误信息，仅用于改善应用体验。

2. 信息使用

  • 播放用户选择的本地音频文件。
  • 显示广告以支持应用开发。
  • 改善应用性能和稳定性。

3. 信息分享

  • 我们不会将用户本地音乐文件或个人数据出售或分享给第三方。
  • 广告 SDK 可能根据匿名信息展示广告，由广告平台处理。

4. 用户选择

  • 您可以关闭应用中的个性化广告（如果广告 SDK 支持）。
  • 可随时删除或管理本地音乐文件。

5. 儿童隐私

  • 本应用面向 13 岁及以上用户。
  • 未满 13 岁儿童不得使用本应用。
